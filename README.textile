h1. twitterボット　簡易管理アプリ

これは予め登録しておいたつぶやきをcron等で定期的にpostするバッチと、つぶやきを予め登録しておくための管理アプリをセットにしたものです。

h2. 動作環境

下記の環境で動作を確認しています。

* CentOS 5.5
* SQLite3
* ruby 1.8.6, rubygem 1.3.1, rails 2.1.0


h2. インストール方法

# Railsアプリが正常に動く状態にしてください
  Railsアプリ部分はscaffoldで生成しただけのものです。
  つぶやきの登録が出来ることを確認してください。

  ユーザ認証はBasic認証を直書きして行っているのでid/pass修正を行って下さい。

# 投稿用バッチスクリプト post.rbおよびgo.sh の設定を行って下さい
  設定変更が必要な箇所については your_ という文字列を書いているので、下記コマンドにより検索し修正を行って下さい。
<pre>
<code>
$ find . -type f|xargs grep your_
</code>
</pre>

  なおoauthの設定も必要です。

# cronによる自動実行を設定してください。

<pre>
<code>
0 0 * * *  /.../go.sh
</code>
</pre>
